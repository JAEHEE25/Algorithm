### [ 문제 풀이 ]
```Java
class Solution {
    public int solution(int[] arrayA, int[] arrayB) {
        //A 공약수 -> B
        int A = getGcd(arrayA);
        if (isDivided(A, arrayB)) {
            A = 0;
        }
        
        //B 공약수 -> A
        int B = getGcd(arrayB);
        if (isDivided(B, arrayA)) {
            B = 0;
        }        
        
        return Math.max(A, B);
    }
    
    boolean isDivided(int v, int[] arr) {
        for (int i = 0; i < arr.length; i++) {
            if (arr[i] % v == 0) return true;
        }
        return false;
    }
    
    int getGcd(int[] arr) {
        int result = arr[0];
        for (int i = 1; i < arr.length; i++) {
            result = gcd(result, arr[i]); 
        }
        return result;
    }
    
    int gcd(int a, int b) {
        if (b == 0) return a;
        return gcd(b, a % b);
    }
}
```
**유클리드 호제법**<br><br>최대 공약수<br>a % b = c<br>b % c = d<br>.<br>.<br>나누는 수가 0이 될때까지<br>-> 재귀로 구현<br><br><br>최소공배수<br>= a*b / 최대공약수<br>
