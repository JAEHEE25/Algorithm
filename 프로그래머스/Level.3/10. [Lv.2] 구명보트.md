### [ 문제 풀이 ]
```Java
import java.util.*;
class Solution {
    public int solution(int[] people, int limit) {
        int n = people.length;
        int answer = n;
        Arrays.sort(people);
        
        int light = 0;
        int heavy = n - 1;
        while (light < heavy) {
            int sum = people[light] + people[heavy];
            if (sum > limit) { //초과하면
                heavy--; //무거운 사람 태우기
            } else {
                light++; //둘 다 태우기
                heavy--;
                answer--; 
            }
        }
        return answer;
    }
}
```
가장 가벼운 사람 + 가장 무거운 사람을 태워야 함
```Java
import java.util.*;
class Solution {
    public int solution(int[] people, int limit) {
        int answer = 0;
        Arrays.sort(people);
        
        int l = 0;
        for (int h = people.length - 1; h >= l; h--) {
            if (people[l] + people[h] <= limit) { //둘이 탈 수 있으면
                l++;
            }
            answer++;
        }
        return answer;
    }
}
```
